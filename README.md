## What This Is

This identicon generator exists because I was bored, curious, and messing around with AI-assisted coding.

I took:

* Some existing Ruby code I had lying around from another project
* A few AI prompts
* A questionable amount of trust in the vibes

And somehow ended up with a single HTML file that generates identicons.

At that point I figured: "Why not throw this on GitHub Pages and let people use it?"

So here we are.

## What This Is Not

Let us be clear:

* This is not a production-grade project
* This is not an example of clean or idiomatic code
* This is not something I would submit to a job interview

The code is atrocious in places. I know it. You know it. The AI definitely knows it.
